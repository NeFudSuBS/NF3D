# NF3D

**3D subtitle creator for Side-by-Side MKV movies.**

Open an SBS MKV, load any subtitle (embedded track or external file), and NF3D automatically analyses the video depth to place each subtitle at the right 3D depth. Full appearance customisation, per-cue editing, and OCR quality review included. Outputs a ready-to-use `.ass` file or muxes it directly into a copy of your MKV.

## First run

The setup wizard runs automatically after install. It checks for the three free tools NF3D needs and can install any that are missing in one click:

- **ffmpeg** — video frame extraction
- **MKVToolNix** — MKV track demuxing and muxing
- **Subtitle Edit** — OCR for image-based subtitle tracks (PGS / VobSub)

If you already have them, the wizard finds them automatically and you go straight to the app.
